# Моніторинг медійного висвітлення соціологічних опитувань

Проєкт, спрямований проаналізувати практики використання та представлення результатів соціологічних досліджень у медійному просторі України.

Проєкт реалізується у форматі регулярного моніторингу, результати якого узагальнюються у квартальних аналітичних звітах. У межах аналізу визначаються ключові замовники та виконавці соціологічних опитувань, їх тематика, виявляються найбільш поширювані опитування та оцінюється дотримання стандартів розкриття інформації відповідно до критеріїв AAPOR.

У межах проєкту застосовуються інструменти штучного інтелекту для автоматизованого виокремлення інформації про соціологічні опитування та ручна верифікація отриманих результатів.
